# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ashishgupta251/C-Programming.git
```

### Open the Project

```bash
cd C-Programming
```

Compile any C program using GCC:

```bash
gcc filename.c -o output
```

Run the executable:

```bash
./output
```
